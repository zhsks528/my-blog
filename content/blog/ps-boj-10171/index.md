## 백준알고리즘 - 10171. 고양이

- 관련된 알고리즘 : .

### 문제

아래 예제와 같이 고양이를 출력하시오.

### 입력

없음

### 출력

고양이를 출력한다.

### 예제

```
Input : 
Output : 
\    /\
 )  ( ')
(  /  )
 \(__)|
```

### 해결

```python
print("""\\    /\\
 )  ( ')
(  /  )
 \\(__)|""")
```
